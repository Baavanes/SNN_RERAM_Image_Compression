## Overview
This project contains the design for (1) a hardware-based SNN using BM Lab's Neuromorphic X1 32x32 IP and (2) The design for an adaptive self-configuring communication fabric.

---
## IMPACT_RERAM_SNN

A Neurosynaptic Core for hand gesture recognition built on a hybrid Artificial/Spiking Neural Network architecture, integrating a single ReRAM-based Neuromorphic X1 NVM macro from BM Labs with the ChipFoundry Caravel SoC Platform.


---
## Adaptive Self-Configuring Communication Fabric

The Adaptive Self-Configuring Communication Fabric automatically detects and configures communication protocols without software intervention.
